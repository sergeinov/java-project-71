```sh
make build
```

### Run
```sh
make run
```

### Run checkstyle
```sh
make lint
```

### Run test
```sh
make test
```

```sh
make report
```

### Update dependencies and plugins
```sh
make update-deps
```

### Clean build
```sh
make clean
```
